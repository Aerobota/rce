addInterface(eTag, iTag, iType, clsName, addr='')

source code 

Add an interface to an endpoint, i.e. a ROS environment or a Robot object.

Parameters:
  • eTag (str) - Tag which is used to identify the endpoint to which the interface should be added; either a container tag or robot ID.
  • iTag (str) - Tag which is used to identify the interface in subsequent requests.
  • iType (str) - Type of the interface. The type consists of a prefix and a suffix.
    • Valid prefixes are: ServiceClient, ServiceProvider, Publisher, Subscriber
    • Valid suffixes are: Interface, Converter, Forwarder
  • clsName (str) - Message type/Service type consisting of the package and the name of the message/service, i.e. 'std_msgs/Int32'.
  • addr (str) - ROS name/address which the interface should use. Only necessary if the suffix of @param iType is 'Interface'.
