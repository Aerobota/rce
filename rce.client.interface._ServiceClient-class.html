Internally used method which should be used to call the service.

Parameters:
  • msg - Request message which should be sent.
  • cb - Callback which should be called to process response. The response will be the first argument and additional arguments passed to the _call method will be passed to the callback.
  • *args - Additional arguments which will be passed to the callback.
