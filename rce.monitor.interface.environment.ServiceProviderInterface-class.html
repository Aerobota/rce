Initialize the ROS-side Interface.

Parameters:
  • owner (rce.environment.Environment) - Namespace to which this interface belongs.
  • uid (uuid.UUID) - Unique ID which is used to identify the interface in the internal communication.
  • clsName (str) - Message type/Service type consisting of the package and the name of the message/service, i.e. 'std_msgs/Int32'.
  • addr (str) - ROS name/address which the interface should use.

_callback(self, request)

source code 

This method is called by the ROS framework when a Service request has arrived. Each call runs in a separate thread and has to block until a response is present, because the return value of this method is used as response to the request.
