onConnect(self, connectionRequest)

 

Callback fired during WebSocket opening handshake when new WebSocket client connection is about to be established.

Throw HttpException when you don't want to accept the WebSocket connection request. For example, throw a `HttpException(httpstatus.HTTP_STATUS_CODE_UNAUTHORIZED[0], "You are not authorized for this!")`.

When you want to accept the connection, return the accepted protocol from list of WebSocket (sub)protocols provided by client or None to speak no specific one or when the client list was empty.

You may also return a pair of `(protocol, headers)` to send additional HTTP `headers`.

:param connectionRequest: WebSocket connection request information. :type connectionRequest: instance of :class:`autobahn.websocket.ConnectionRequest`
