Initialize the Robot Client.

Parameters:
  • reactor (twisted::reactor) - Reference to the twisted reactor used in this robot process.
  • masterIP (str) - IP address of the Master process.
  • masterPort (int) - Port which should be used to authenticate the user with the Master process.
  • commPort (int) - Port where the server for the cloud engine internal communication is listening for incoming connections.
  • extIP (str) - IP address of network interface used for the external communication.
  • extPort (int) - Port where the server for the external communication is listening for websocket connections.
  • loader (rce.util.loader.Loader) - Object which is used to load Python modules from ROS packages.
  • converter (rce.util.converter.Converter) - Converter which takes care of converting the messages from JSON to ROS message and vice versa.
