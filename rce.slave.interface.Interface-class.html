send(self, msg, msgID, protocol, remoteID)

source code 

This method is used to send a message to the endpoint.

Don't overwrite this method; instead overwrite the method _send. If the interface does not overwrite the method _send, it is assumed that the interface does not support this action and an InternalError is raised when send is called.

Parameters:
  • msg (str) - Message which should be sent in serialized form.
  • msgID (str) - Message ID which is used to match response message.
  • protocol (rce.slave.protocol._Protocol) - Protocol instance through which the message was sent.
  • remoteID (uuid.UUID) - Unique ID of the Interface which sent the message.
