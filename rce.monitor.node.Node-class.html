Initialize and start the Node.

Parameters:
  • owner (rce.environment.Environment) - Environment in which the node will be created.
  • pkg (str) - Name of ROS package where the node can be found.
  • exe (str) - Name of executable (node) which should be launched.
  • args (str) - Additional arguments which should be used for the launch. Can contain the directives $(find PKG) and/or $(env VAR). Other special characters such as '$' or ';' are not allowed.
  • name (str) - Name of the node under which the node should be launched.
  • namespace (str) - Namespace in which the node should be started in the environment.
