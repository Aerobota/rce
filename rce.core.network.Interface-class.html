registerRemoteID(self, protocol, remoteID)

source code 

Register an interface (using its unique ID) with this interface, which means that this and the indicated interface will be connected.

Parameters:
  • protocol (rce.master.network.Protocol) - Protocol which provides the connection to the other side, i.e. the other endpoint. (Could also be the loopback protocol which would mean that both interfaces are in the same endpoint.)
  • remoteID (uuid.UUID) - Unique ID of the interface at the other side of the connection, i.e. in the other endpoint.

None.
