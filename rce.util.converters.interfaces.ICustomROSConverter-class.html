encode(rosMsg)

Generate dictionary from a ROS message.

Parameters:
  • rosMsg (genpy.message.Message) - The ROS message instance which should be converted.
Returns: {}
Dictionary containing the parsed message. The basic form does map each field in the ROS message to a key / value pair in the returned data dict. Binaries are added as StringIO instances.
