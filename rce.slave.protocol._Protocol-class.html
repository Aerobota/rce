sendMessage(self, interface, msg, msgID, remoteID=None)

source code 

Send a message received from an Interface to the other side.

Parameters:
  • interface (rce.slave.interface.Interface) - Interface which wants to send the message.
  • msg (str) - Message which should be sent.
  • msgID (uuid.UUID) - Unique ID which can be used to find a correspondence between request / response message.
  • remoteID (uuid.UUID) - If the remote ID is supplied than only this Interface will receive the message, regardless of additional interfaces which might be registered.
