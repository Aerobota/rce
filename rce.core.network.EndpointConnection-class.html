source code 

Internally used method which is part of a callback chain. Its task is to verify that both endpoints are ready for the connection attempt. In case both signal readiness the address of the designated server endpoint is retrieved.

Parameters:

_connectPrepError(self, failure, authenticator)

source code 

Internally used method which is part of an errback chain. Its task is to signal to the authenticator that the connection can not be attempted by cancelling the Deferred.

_validate(self, result)

source code 

Internally used method which is part of a callback chain. Its task is to verify that both sides of the connection could be authenticated. In case both signal success the corresponding protocols are registered.
