addInterface(self, iTag, iType, clsName)

source code 

Add an interface to the Robot object.

Parameters:
  • iTag (str) - Tag which is used to identify the interface in subsequent requests.
  • iType (str) - Type of the interface. The type consists of a prefix and a suffix.
    • Valid prefixes are: ServiceClient, ServiceProvider, Publisher, Subscriber
    • Valid suffixes are: Converter, Forwarder
  • clsName (str) - Message type/Service type consisting of the package and the name of the message/service, i.e. 'std_msgs/Int32'.
